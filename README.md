This project is a web-based application developed to simplify and organize day-to-day operations in a clinic. It allows efficient management of patients, appointments, and medical records through a structured and user-friendly interface.

---

## Overview

The system is designed to reduce manual work in clinics by digitizing processes such as patient registration, appointment booking, and record maintenance. It supports multiple user roles to ensure that each user has access only to the features relevant to them.

---

## Key Functionalities

- **User Authentication**
  - Separate login access for administrators, doctors, and patients
  - Controlled access based on user roles

- **Appointment Management**
  - Patients can request or book appointments
  - Doctors can view and manage their schedules

- **Patient Records**
  - Stores patient details and medical history
  - Enables easy retrieval and updates

- **Doctor Dashboard**
  - View assigned patients and appointments
  - Update treatment details and observations

---

## Technology Used

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS
- **Database:** SQLite
- **Version Control:** Git and GitHub
